About Eclipse

Eclipse is a web browser project focused on doing one thing well: giving users a browsing experience that actually respects their time, attention, and hardware. Most browsers have become bloated, intrusive, or optimized for someone else's agenda. Eclipse is aiming to fix that.

The project is currently in active development, built using modern web technologies with a focus on performance, clean UI design, and keeping the user in control. It's an ambitious project — and honest about that.

Eclipse is also a natural fit for the Phantom Labs ecosystem — as NorthByte builds out more tooling and ClassFlow develops its platform, Eclipse becomes an interesting integration point for the future.

What We're Focused On

⚡

Performance

A browsing experience that feels noticeably faster and lighter than what you're used to.

🔒

Privacy-Aware

Designed to minimize unnecessary data collection. Your browsing stays yours.

🎨

Clean Interface

A minimal UI that gets out of the way. Everything you need, nothing you don't.

📑

Tab Management

Smarter tab and session handling for people who actually have a lot open at once.
